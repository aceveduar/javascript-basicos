
## Variables en JavaScript

Hay dos estados de una variable

1. Declarar una variable 

```jsx
var edad;
```

1. Inicializar una variable

```jsx
edad = 28;
```

Variables de arrays

```jsx
let elementos = ["Computadora", "Celular", "Tableta"]
```

Objetos
```jsx
let persona = {
	nombre: "Eduardo",
	edad: 30
}

persona();
```
